# Dungeon-Descend
O projeto "Dungeon Descend" consiste em um jogo digital do gênero RPG Tático de Turnos (TRPG) com elementos de Roguelike, desenvolvido primariamente para computadores. O jogo consiste em formar um grupo de heróis, explorar salas e enfrentar inimigos num combate estratégico posicionado em um grid 2D isométrico.
